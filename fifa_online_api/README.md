# FIFA Online4 match finder
golang 연습을 위해 만들어보았습니다. FIFA online4 API를 이용한 경기정보 탐색기입니다.

```bash
https://developers.nexon.com/fifaonline4/guides  
```
위의 guide를 참고해서 API token을 발급받은뒤 사용하시면 됩니다(token이 유출되지 않도록 주의)
